# 情侣复盘 Skill · 快速开始

**你只需20分钟，就能完成一次专业的对抗式关系复盘。**

---

## 你需要准备什么

- 一段想复盘的冲突（具体，不要泛泛的）
- 聊天记录（可选，没有也能做）

---

## 完整流程

```
① 上传记录（5分钟）
    └─ 粘贴聊天记录 / 发文件路径 / 直接描述
         ↓
② 补充信息（2分钟）
    └─ 冲突时间、结果、你想要什么
         ↓
③ 描述冲突（3分钟）
    └─ 具体事件，不要抽象
         ↓
④ 对抗式对话（6分钟）
    └─ 两个子代理模拟你们的真实对话
         ↓
⑤ 论文复盘总结（5分钟）
    └─ Paper A/B/C 理论分析 + 可执行方案
```

---

## 如何开始

把这篇文稿发给我：

```
我想用情侣复盘 Skill 做一次复盘。
[粘贴聊天记录 / 文件路径 / 文字描述]
这是我的复盘请求。
```

---

## 上传记录的方式

| 方式 | 怎么做 |
|------|--------|
| **直接粘贴** | 复制微信聊天记录发给我，格式不限 |
| **文件路径** | 微信解密后的聊天记录文件路径 |
| **没有记录** | 直接描述冲突经过，也能做 |

---

## 你会得到什么

| 输出 | 说明 |
|------|------|
| 《复盘事件时间线》 | 客观记录冲突经过 |
| 《对抗式对话》 | 两个子代理模拟的真实对话还原 |
| 《参谋批评报告》 | 从对方视角攻击你的每一个行为 |
| 《论文复盘报告》 | Paper A/B/C 理论分析 |
| 《复盘报告》 | 裁判打分 + 核心矛盾 + Top3雷点 |
| 《动态黑名单》 | 新发现的雷点追加记录 |
| 《改进方案》 | 具体可执行的改进（最多3条） |

---

## 怎么描述冲突（重要）

**好的描述：**
> "上周五晚上我说要加班，他连发了十几条消息问我和谁在一起，我没回，他第二天一整天没理我"

**不好的描述：**
> "我们沟通有问题"
> "他总是这样"

越具体，我能分析得越准确。

---

## 注意事项

- **选一个具体冲突**，不要同时复盘很多件事
- **情绪平复后再来**，不要在气头上
- **和对方一起看报告**，复盘是为了两个人一起变好
